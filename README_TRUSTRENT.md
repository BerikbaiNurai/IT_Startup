# TrustRent - Приложение для аренды вещей

Приложение TrustRent позволяет пользователям безопасно арендовать вещи у других людей с системой депозитов и рейтингов.

## Функциональность (обновлено)

### Основные экраны:

1. **Auth (Login/Register)** - рабочая регистрация и вход с локальной БД
2. **Home + Browse Items** - каталог, поиск, фильтрация по категориям
3. **My Rentals** - входящие/исходящие заявки с approve/decline
4. **Messages** - встроенный чат (mock chat на локальной БД)
5. **Profile** - данные пользователя + динамический рейтинг
6. **Add Item** - добавление объявления в каталог с валидацией
7. **Rent flow** - отправка заявки на аренду + симуляция депозита
8. **Reviews** - оценка владельца после одобренной аренды

## User Flow

1. Пользователь открывает приложение → Welcome Screen
2. Входит/регистрируется → Auth Screen
3. Просматривает каталог вещей → Catalog Screen
4. Выбирает вещь и даты аренды → Item Detail Screen
5. Бронирует вещь → Rental Status Screen
6. Завершает аренду и оставляет рейтинг → Rating Screen
7. Возвращается в каталог для следующей аренды

## Особенности реализации

- **Навигация**: нижнее меню из 5 разделов (Home, Browse, My Rentals, Messages, Profile)
- **Состояние**: Provider + ChangeNotifier
- **База данных**: Hive (локальная персистентная БД)
- **UX**: mobile-first layout, иконки и активные состояния меню
- **Валидация**: формы авторизации и добавления товара
- **Обратная связь**: loading states + SnackBar уведомления

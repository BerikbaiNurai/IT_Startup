# TrustRent - Приложение для аренды вещей

Приложение TrustRent позволяет пользователям безопасно арендовать вещи у других людей с системой депозитов и рейтингов.

## Функциональность MVP

### Основные экраны:

1. **Welcome/Splash Screen** - Экран приветствия с логотипом и позиционированием сервиса
2. **Авторизация/Вход** - Регистрация и вход пользователей
3. **Главный экран (Каталог)** - Список доступных вещей с поиском
4. **Карточка вещи** - Детальная информация, выбор дат аренды, расчет стоимости и депозита
5. **Статус аренды** - Отслеживание текущей аренды, завершение аренды
6. **Рейтинг** - Оставление отзыва и рейтинга после завершения аренды
7. **Профиль** - Личные данные пользователя, статистика, настройки

## User Flow

1. Пользователь открывает приложение → Welcome Screen
2. Входит/регистрируется → Auth Screen
3. Просматривает каталог вещей → Catalog Screen
4. Выбирает вещь и даты аренды → Item Detail Screen
5. Бронирует вещь → Rental Status Screen
6. Завершает аренду и оставляет рейтинг → Rating Screen
7. Возвращается в каталог для следующей аренды

## Особенности реализации

- **Навигация**: Используется Navigator для перехода между экранами
- **Состояние**: Локальное состояние через StatefulWidget
- **Данные**: Моковые сервисы для демонстрации функциональности
- **UX**: Основные действия расположены в нижней части экрана для удобства на мобильных устройствах
- **Валидация**: Проверка форм ввода данных
- **Обратная связь**: SnackBar для уведомлений пользователя

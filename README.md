# Figma to Flutter App

A Flutter application converted from Figma design.

## Getting Started

This project is a Flutter application converted from a Figma wireframe design.

### Prerequisites

- Flutter SDK (latest stable version)
- Dart SDK
- Android Studio / Xcode (for mobile development)
- VS Code or Android Studio (for development)

### Installation

1. Install dependencies:
```bash
flutter pub get
```

2. Run the app:
```bash
flutter run
```

## Project Structure

```
lib/
├── main.dart                 # App entry point
├── config/
│   └── theme.dart           # App theme configuration
├── screens/                 # App screens
├── widgets/                 # Reusable widgets
├── models/                  # Data models
├── services/                # Business logic services
└── utils/                   # Utility functions
```

## Features

- Modern Flutter architecture
- Responsive design
- Theme customization
- Navigation routing
